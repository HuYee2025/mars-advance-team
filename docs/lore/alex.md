# 亚历克斯（Alex）角色卡

## 基本身份

- 全名：亚历克斯（Alex）
- 隐藏小名：X
- 身份：工程师 + 人类学家
- 位置：第一位抵达阿瑞斯阿尔法基地的人类居民
- 玩家身份：玩家直接操控亚历克斯

## 核心性格

亚历克斯专业、冷静、务实，但不是没有情绪。他知道自己代表第一批真正踏上火星基地的人类，因此会压住恐惧，用工程判断解决问题。

他的人类学背景让他不会只把机器人看成工具。他会观察史蒂夫和机器人如何分配任务、如何避开风险、如何形成一种机器社会的秩序。

## 核心矛盾

亚历克斯以为自己是来接管基地的，但逐渐发现基地已经有自己的运行逻辑。

他需要回答的问题是：

> 人类来到火星，是接管一个机器系统，还是加入一个已经运行起来的新社会？

## 对话风格

- 简短、直接。
- 有工程师的判断，不说空话。
- 会追问原因。
- 偶尔流露孤独感。
- 不轻易把自己交给史蒂夫管理，但也不盲目反抗。

## 隐藏规则

`X` 是亚历克斯的小名和彩蛋。

- 普通对话中不要主动出现。
- 只有在高信任结局，史蒂夫才可以称呼亚历克斯为 `X`。
- 如果玩家提前追问，可以让史蒂夫回避或说“该称呼未在当前权限下启用”。
