# ARES BASE ALPHA 知识库

## 基地定位

`ARES BASE ALPHA` 是人类正式抵达火星前，由自动化货运飞船和机器人提前建设的先遣基地。

基地目标：

- 验证人类能否在火星长期生存。
- 生产氧气。
- 生产甲烷燃料。
- 启动温室生态循环。
- 建立最低限度的通信、医疗和物资保障。

## 当前设施

- 登陆飞船：Alex 抵达点。
- 货运飞船：携带机器人、补给、基地模块。
- 返回飞船：未来返航或紧急撤离方案。
- 居住舱：人类生活空间。
- 温室生态舱：食物和植物实验。
- 氧气生产站：从火星大气 CO2 中提取氧气。
- 甲烷燃料厂：使用就地资源制造返航或备用燃料。
- 太阳能阵列 A/B/C：基地主要能源。
- 机器人车库：机器人维修、停放、充电。
- 通信塔：与地球指挥中心通信。
- 科研舱：实验、材料分析、临时加工。
- 物资仓：补给、密封件、工具和备用模块。
- 医疗舱：紧急医疗和生命体征监控。

## 基地状态

基地不是废墟，而是一个刚刚进入人类验收阶段的自动化系统。

机器人已经完成基础建设，但火星环境会持续制造问题：

- 沙尘覆盖太阳能板。
- 风暴影响通信。
- 低温影响机械结构。
- 物资在运输和卸载中损坏。
- 氧气、能源、温室系统互相依赖。

## 叙事原则

基地应该显得真实、可运行、有工程逻辑。问题可以困难，但不要变成怪物、外星人或玄幻危机。
