# 火星生存知识库

## 火星环境

火星表面寒冷、干燥、大气稀薄。人类不能直接暴露在户外环境中。

游戏中重点表现：

- 氧气短缺。
- 能源不足。
- 沙尘暴。
- 通信延迟。
- 物资损坏。
- 设备维护困难。

## 氧气

火星大气主要是 CO2。基地可以通过就地资源利用系统，把 CO2 转化为氧气。

游戏表现：

- 氧气生产站需要稳定供电。
- CO2 进气口可能被沙尘影响。
- 氧气站异常不一定是泄漏，也可能是供电或压缩机问题。

## 能源

太阳能是早期基地的主要能源，但火星沙尘会降低效率。

游戏表现：

- 沙尘覆盖太阳能板。
- 阵列角度需要校准。
- 风暴期间必须切换低功耗模式。

## 温室

温室不是装饰，而是长期生存系统的一部分。

游戏表现：

- 水循环、温控、补光都需要稳定。
- 种植选择可以影响长期生存和剧情反馈。

## 通信

地球和火星之间有明显通信延迟。远程命令不能实时解决所有问题。

游戏表现：

- Alex 必须做现场判断。
- Mother 会基于本地数据给出安全建议。
- 地球命令可能正确，但不一定适合当前情况。
