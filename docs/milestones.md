# 里程碑

## Phase 1 - 1.0 可试玩版本（已基本完成）

- 建立 Vite + Three.js 工程。
- 生成火星地形、基地建筑、登陆器、机器人和沙尘。
- 支持桌面键盘/鼠标与移动端摇杆。
- 建筑靠近显示标签。
- 主线任务闭环：熟悉基地、氧气异常、太阳能阵列、机器人车库。
- 写实半身像对话系统。
- Mother 呼叫。
- 当前说话人放大提示。
- 氧气背包消耗和补给。
- 地图雷达、当前任务、目标高亮。
- 移动端按钮、右上角地图入口、小屏封面隐藏控制按钮。
- 01/02 飞船升降梯可用。
- 删除开发比例尺模型。

## Phase 2 - 视觉增强

- 增加更丰富的地貌：裂谷、岩层、基地道路、脚印或车辙。

- 加入日落光照、低频火星风声和基地机械环境音。
