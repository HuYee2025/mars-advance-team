# 当前交接记录

更新时间：2026-06-27

## 用途

这是新对话进入项目时的快速交接入口。若同一对话发生约 3 次自动压缩，先检查本文件和相关主管文档是否完整；完整后建议开启新对话继续。

新对话默认先读：

1. `AGENTS.md`
2. `docs/project-management.md`
3. `docs/production-blueprint.md`
4. `docs/dev-log/current-handoff.md`
5. `docs/tech-plan.md`
6. `docs/decision-log.md`

## 最近已完成的重点

- 首页火星日期改为 ARES 火星历动态日期：以 `2026-06-27 00:00 Asia/Shanghai` 对应 `2050 年 / 第 30 火星日` 为锚点，每 `88775.244` 秒推进一个 sol，每 `669` sols 进位一年。
- 首页语言切换只在首页可用；进入游戏后隐藏。按钮已改为右上角约 `24px` 的立体自转国旗小球：中文界面显示美国国旗，英文界面显示中国国旗。
- 首页两个主按钮去掉浏览器蓝色焦点框，改为和游戏内顶部按钮一致的暗底、棕金边框、红色静态发光交互状态。
- 右上角金币和积分改为按钮化 HUD：金币初始 `$`，积分初始 `0`，获得后用三位数字显示。
- `V`、`R`、`F` 顶部按钮统一视觉：关闭态半透明深色底；信息打开时静态红光；只有 `R` 有新任务时动态发光。
- 小屏任务提示改为 `R` 按钮开关；移动端 `R` 与 `F` 互斥，桌面端可以同时打开。
- 小屏剧情对话框固定高度并上移；正文最多三行，超出自动分页；底部 `继续/选项` 按钮固定在面板底部按钮区，不随正文高度跳动。
- 移动端双选项对话使用 `A/B` 短按钮，完整选项文字写入正文。
- 对话按钮点击、翻页和确认播放与首页“进入基地”一致的短促 UI beep。
- 头盔灯/照明功能已移除。
- 飞行作弊模式已调整：下降可落地并退出飞行背包；飞行中取消走路动作，改为腿部轻微摆动；可落在飞船高处走廊等可行走表面。
- 太阳能阵列 A/B/C 已移动到更靠近 `03 飞船 返回飞船` 一侧的首页亮面区域，源码坐标为 `(-22,52) / (-4,52) / (14,52)`，并同步管线、维修机器人、碰撞体、雷达和互动点。
- 新增 `ARES Starlink-M` 火星低轨通信星座：进入游戏后天空直接存在第一批 10 颗星链卫星，之后每 24 个火星日新增 10 颗，上限 120 颗；不做释放动画。
- 星链卫星有可见实体模型和微光点，轨道高度按火星真实半径比例换算；通信塔校准后任务提示和雷达显示轨道链路状态。
- 缩放枪可锁定并命中每颗星链卫星模型；每颗卫星首次命中奖励隐藏发现积分。
- 缩放枪也可锁定飞过近火轨道的那颗石质流星本体；命中后流星会按缩放枪倍率变化 60 秒，首次命中奖励隐藏发现积分。其它两颗远处天体不参与缩放枪判断。

## 下次继续时优先检查

- 首页语言国旗小球的视觉是否满意，尤其是自转速度、明暗面和国旗识别度。
- 首页视角下太阳能阵列是否落在用户标注的亮面区域，是否仍避开温室、03 飞船和基地建筑。
- 移动端对话面板在不同文字长度下，底部按钮是否完全稳定。
- 星链卫星在桌面和移动端天空中是否像远处轨道微光，缩放枪能否锁定并击中奖励。
- 近火掠过流星出现时，缩放枪能否锁定这一颗流星本体并触发一次性积分奖励；其它两颗远处天体不应被锁定。
- 长对话如再次接近 3 次压缩，先检查本文件是否需要更新，再建议开启新对话。
