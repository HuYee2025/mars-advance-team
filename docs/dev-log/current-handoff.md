# 当前交接记录

更新时间：2026-06-27

## 用途

这是新对话进入项目时的快速交接入口。若同一对话发生约 3 次自动压缩，先检查本文件和相关主管文档是否完整；完整后建议开启新对话继续。

新对话默认先读：

1. `AGENTS.md`
2. `docs/project-management.md`
3. `docs/production-blueprint.md`
4. `docs/dev-log/current-handoff.md`
5. `docs/tech-plan.md`
6. `docs/decision-log.md`

## 最近已完成的重点

- 首页火星日期改为 ARES 火星历动态日期：以 `2026-06-27 00:00 Asia/Shanghai` 对应 `2050 年 / 第 30 火星日` 为锚点，每 `88775.244` 秒推进一个 sol，每 `669` sols 进位一年。
- 首页语言切换只在首页可用；进入游戏后隐藏。按钮已改为右上角约 `24px` 的立体自转国旗小球：中文界面显示美国国旗，英文界面显示中国国旗。
- 首页已补首屏 critical CSS，避免本地预览服务器或慢网打开时短暂出现白底黑字的未样式化 HTML；语言切换小球改为在 24px 圆形内部滚动 background-position，避免 Safari 把旗面动画渲染成小圆柱。
- 首页两个主按钮去掉浏览器蓝色焦点框，改为和游戏内顶部按钮一致的暗底、棕金边框、红色静态发光交互状态。
- 首页标题样式微调：中文首页大标题字重提高；英文首页在 `520px` 以下小屏按三行显示为 `Mars Advance` / `Team` / `The First Human`，避免字母溢出屏幕。
- 首页入口支持三种方式：`W/S` 或方向键上下选择后用 `E/Enter/Space` 确认；默认选中“进入基地”，所以首页直接按 `E` 可进基地，按 `Q` 可进故事概要；鼠标悬停按钮也会切换当前选中项，再用 `E/Enter` 确认。
- 右上角金币和积分改为按钮化 HUD：金币初始 `$`，积分初始 `0`，获得后用三位数字显示。
- `V`、`R`、`F` 顶部按钮统一视觉：关闭态半透明深色底；信息打开时静态红光；只有 `R` 有新任务时动态发光。
- 小屏任务提示改为 `R` 按钮开关；移动端 `R` 与 `F` 互斥，桌面端可以同时打开。
- 小屏剧情对话框固定高度并上移；正文最多三行，超出自动分页；底部 `继续/选项` 按钮固定在面板底部按钮区，不随正文高度跳动。
- 移动端双选项对话使用 `A/B` 短按钮，完整选项文字写入正文。
- 对话按钮点击、翻页和确认播放与首页“进入基地”一致的短促 UI beep。
- 头盔灯/照明功能已移除。
- 飞行作弊模式已调整：下降可落地并退出飞行背包；飞行中取消走路动作，改为腿部轻微摆动；可落在飞船高处走廊等可行走表面。
- 太阳能阵列 A/B/C 已移动到更靠近 `03 飞船 返回飞船` 一侧的首页亮面区域，源码坐标为 `(-22,52) / (-4,52) / (14,52)`，并同步管线、维修机器人、碰撞体、雷达和互动点。
- 新增 `ARES Starlink-M` 火星低轨通信星座：进入游戏后天空直接存在第一批 10 颗星链卫星，之后每 24 个火星日新增 10 颗，上限 120 颗；不做释放动画。
- 星链卫星按两串编队飞过天空，每串 5 颗；视觉上是比背景星点略亮、略大的圆形微光点，不再显示成方形亮块。
- 星链卫星有很小的可见实体模型和不可见缩放枪命中体，轨道高度按火星真实半径比例换算；通信塔校准后任务提示和雷达显示轨道链路状态。
- 缩放枪可锁定并命中每颗星链卫星模型；每颗卫星首次命中奖励隐藏发现积分。
- 缩放枪也可锁定近火轨道的那颗石质流星本体；该近火流星已改成绕火星的固定圆形近火轨道，轨道半径恒定为 `PLANET_RADIUS + 150`，也就是高出火星地表约 `150` 游戏单位，约等于三艘飞船叠加高度；它从玩家初始基地上空轨道点开始，每现实约 `90` 秒绕行一圈并回到同一上空轨道点，不再使用容易产生贴地错觉的掠过高度曲线。近火流星头部不再使用蓝色气体罩、蓝色气体尾巴或蓝色自发光，本体材质为不透明深灰石质。近火流星会在雷达上显示为蓝色点；命中后流星会按缩放枪倍率变化 60 秒，首次命中奖励隐藏发现积分。其它两颗远处天体不参与缩放枪判断。
- 游戏正式对话已排查并移除所有三选项节点，最多只保留两个选择，以匹配桌面端 `Q/E` 二选一操作。
- 雷达标记改为分类清晰：任务目标黄色，并带一圈圈放大渐隐的黄色雷达光圈；氧气补给绿色；金币为小 `$` 金币样式且不发光。
- 雷达外圈内侧新增 `N/E/S/W` 方位环，随玩家视线左右旋转实时更新；坐标下方新增更明显的 `方位 000°` / `HDG 000°` 朝向读数。
- 神秘物体、未知生命、黑色方碑和 Elon 只在未发现或未对话前显示 `?`；发现或对话后雷达移除 `?`，改用对应分类圆点和真实名称。
- 长按 `F` 改为临时全屏雷达：背景虚化、游戏世界继续运行，但键盘/人物操作焦点全部交给雷达，人物不可移动也不可触发普通互动；松开 `F` 后恢复到长按前的普通显示状态。全屏雷达期间 `Q` 缩小显示更大区域，`E` 放大显示更小区域，圆点和文字随放大轻微变大。
- 体能/氧气规则改为探索导向：走路时体能基础消耗为氧气数值下降速率的 0.5 倍，并乘以 `1.75` 手感系数；加速时体能为走路 2 倍、氧气为走路 1.5 倍；跳跃移动体能为走路 0.5 倍；原地站立缓慢恢复体能但氧气继续下降；进入或离开居住舱满氧满体能。
- 首次把火星足球踢进球门会一次性解锁相机奖励，弹出三行奖励提示：`恭喜！你获得了一台相机`、`随时按 G 键可以使用相机`、`照片会展示在居住舱的墙上`。相机不建模；按 `G` 打开/收起取景框，取景时人物不移动，`A/D` 转向，`W/S` 缩放，`E/Enter` 拍照。
- 居住舱内部新增 7 格实体照片墙；拍照后缩略图写入照片墙，最多保留 7 张，超过后替换最旧照片。进入居住舱且已有照片时可按 `E` 查看照片墙大图；大图查看中 `A/D` 切换，`W` 放大，`S` 恢复，`Q` 退出，`E/Enter` 触发当前照片 PNG 下载。浏览器版本不静默保存到桌面。
- 英文版补齐剧情对白、机器人对白、Elon 对话、缩放枪提示和建筑互动说明的精确英文映射；`src/main.ts` 新增 `runtimeEnglishTexts`，`scripts/i18n-audit.mjs` 已纳入该表检查。自定义扫描确认英文模式本地化后不再残留中文字符串。
- `public/story-overview.html` 故事概要独立页已补英文翻译；首页点击故事概要时会带上当前语言参数，故事页也会读取 `localStorage["mars.language"]` 兜底。Playwright 已验证 `story-overview.html?lang=en-US` 首屏、正文和章节索引为英文。
- 故事概要页首屏标题已按视口居中；左上角返回主页按钮改为首页同系红色发光交互，不再使用蓝色 hover/focus。

## 下次继续时优先检查

- 首页语言国旗小球的视觉是否满意，尤其是自转速度、明暗面和国旗识别度。
- 首页视角下太阳能阵列是否落在用户标注的亮面区域，是否仍避开温室、03 飞船和基地建筑。
- 移动端对话面板在不同文字长度下，底部按钮是否完全稳定。
- 星链卫星在桌面和移动端天空中是否像一串远处轨道微光，单颗不要明显大于背景星点太多；缩放枪能否锁定并击中奖励。
- 近火圆形轨道流星出现时，缩放枪能否锁定这一颗流星本体并触发一次性积分奖励；其它两颗远处天体不应被锁定。
- 对话流程中不应再出现第三选项；雷达金币、任务目标、氧气补给三者应能明显区分。
- 长按 `F` 的全屏雷达是否能保持世界动态刷新但锁住人物操作，松开是否恢复；全屏雷达中 `Q/E` 是否调整雷达范围。
- 体能/氧气新数值需要实际试玩手感：跳跃探索应明显更省体能，但不能让氧气路线规划失效。
- 相机奖励需要实际试玩确认：首次足球进球是否只弹一次三行奖励框；`G` 取景、`E/Enter` 拍照、居住舱照片墙查看和 `E/Enter` 下载是否符合预期；7 张上限替换旧照片是否自然。
- 后续新增剧情对白或运行时提示时，优先补 `runtimeEnglishTexts`，再运行 `pnpm i18n:audit` 检查。
- 后续修改故事概要页时，需要同步维护 `public/story-overview.html` 内置英文映射，并用 `?lang=en-US` 做一次真实页面检查。
- 长对话如再次接近 3 次压缩，先检查本文件是否需要更新，再建议开启新对话。
