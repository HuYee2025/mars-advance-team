## 已修改代码

- `src/world.ts`
  - 新增 `fufuRescueSite`。
  - 福福初始位置设在坠毁飞船残骸旁。

- `src/main.ts`
  - 创建福福模型并加入场景。
  - 给福福添加场景标签。
  - 新增 `fufuRescued`、`activeFufu`、`fufuSpeed` 状态。
  - 玩家靠近未救助的福福时，显示 `按 E 安抚 福福`。
  - 按 E 后福福进入已救助状态，并开始跟随 Alex。
  - 福福跟随时保持在 Alex 后侧，距离过远会追上。
  - Alex 进入居住舱、温室、飞船内舱或电梯时，福福暂时隐藏。
- 雷达打开后按 `O`，未救助的福福会作为 `未知生命迹象` 问号显示。
- 救助福福后，雷达未知问号消失。
- 未救助时，福福会在坠毁飞船旁左右巡看；Alex 靠近后会转身看向 Alex。
- 救助后，福福不会固定贴在 Alex 身后，而是在 Alex 周围灵活游走；距离拉开后再主动追上。
- 福福奔跑动画已降低腿部摆幅和频率，减少抽动感。
- 修复雷达未知点镜像问题：坠毁火箭在星球背面时，地图必须使用对象真实世界位置，而不是用 `x/z` 重新生成正面 `planetNormal`。否则会在正面同坐标出现不存在的问号。
- 福福被救助并开始跟随 Alex 后，不再显示名字标签；未救助或未关联状态下才显示。
- 福福外观已改成火星宇航猫：透明玻璃头盔、白色简洁宇航服、可见猫头、四肢和尾巴；整体比例放大，方便看清“猫穿宇航服”的轮廓。

## 当前玩法

1. 打开游戏。
2. 按 `M` 打开雷达。
3. 按 `O` 显示未知点。
4. 朝问号方向寻找坠毁飞船残骸附近的福福。
5. 靠近后按 `E` 安抚福福。
6. 福福开始跟随 Alex。
