# 1.0 实现交接记录

更新时间：2026-06-23

## 当前版本定位

当前版本是《火星先遣队：第一位人类》的 1.0 可试玩版本。

核心体验已经形成：

- 火星小星球基地探索。
- 低多边形 3D 场景。
- 写实半身像对话演出。
- Mother 呼叫与角色对话。
- 主线任务：熟悉基地、氧气异常、太阳能阵列、机器人车库。
- 地图雷达、任务提示、目标高亮。
- 宇航服氧气背包消耗与补给。
- 移动端虚拟摇杆和按钮操作。

## 已实现功能

### 场景与设施

- 火星球体地形、星空、沙尘、风暴效果。
- 登陆飞船、货运飞船、返回飞船。
- 居住舱、温室生态舱、氧气生产站、甲烷燃料厂、机器人车库、通信塔、科研舱、物资仓、医疗舱。
- 太阳能阵列 A/B/C。
- 机器人、火星车巡逻。
- 删除开发用比例参考尺模型。
- 修复太阳能面板支撑杆穿模问题。

### 飞船与建筑交互

- 01 飞船升降梯可使用。
- 02 飞船升降梯已接入同一套交互逻辑。
- 飞船内舱改为较暗的桶状结构，镜头后退，能看到上下弧形舱体。
- 居住舱、温室可进入/离开。

### 对话系统

- 对话采用写实半身像，不使用低多边形模型。
- Alex、Mother、维修机器人使用半身像。
- 对话框为半透明科幻液晶面板。
- 当前说话人会放大、提亮；非说话人缩小弱化。
- 角色名字牌放在人物前方、对话框上方，避免遮挡文字。
- 对话选择不使用数字键。
- 对话选择统一使用左右键切换、空格确认。

### 互动规则

- 靠近可互动对象后，按 `E` 进入互动或打开互动选择。
- 如果同时有两个互动目标，显示两个选项文字。
- 互动选择只用左右键切换、空格确认。
- 不使用鼠标选择互动。
- 尽量减少显性操作说明，只显示可做的事。

### HUD 与地图

- 左侧 HUD 分为“基地”和“人员”。
- 基地指标：制氧量、能量、机器人。
- 人员指标：Alex 状态、氧气、体能。
- 将“自养量”统一改为“制氧量”。
- 右上角增加圆形 `M` 按钮。
- 左上角 `V` 按钮改为同样的圆形。
- `V` 与 `M` 左右对称。
- 当前任务移到地图雷达下方，宽度跟雷达一致。
- 当前任务目标在雷达上变绿、发光。
- 小屏幕下点击雷达可进入全屏地图。
- 小屏幕右下角地图按钮已隐藏，地图入口统一由右上角 `M` 提供。

### 移动端

- 小屏幕封面页隐藏底部移动控制按钮。
- 进入基地后才显示摇杆、互动、跳、照明按钮。
- 移动端保留虚拟摇杆。
- 右下角只保留照明、跳、互动。

### 封面与视觉

- 封面主标题固定两行：
  - 火星先遣队
  - 第一位人类
- 去掉标题冒号。
- 副标题为中文。
- “进入基地”按钮改为飞船舷窗风格：
  - 深色星空。
  - 下方红色星球弧面。
  - 金属边框。
- favicon 改为独立 `/favicon.svg` 文件。

### 操作指南

- 操作指南由 `Backquote` 键控制，也就是 ESC 下方的 `· / ~` 键。
- 按住显示，松开隐藏。
- 不再使用 `F1`。
