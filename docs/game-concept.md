# 火星殖民 Demo 设定

## 核心概念

`ARES BASE ALPHA` 是第一批自动化货运飞船抵达火星后部署的先遣基地。人类还没有大规模定居，基地主要由机器人维护，任务是验证氧气生产、甲烷燃料制造、温室种植和能源供给。

## 场景组成

- 红色弧面火星地形：陨石坑、岩石、沙尘、薄雾。
- 登陆飞船：垂直着陆的星际货运飞船，但不复制任何真实品牌外观。
- 居住舱：白色密封模块，代表早期宇航员生活空间。
- 温室：透明穹顶、植物床和绿色补光。
- 氧气生产站：参考 MOXIE/ISRU 概念，用 CO2 制氧。
- 甲烷燃料厂：参考 Sabatier 反应概念，生产返航燃料。
- 太阳能阵列：基地主要能源视觉符号。
- 机器人车库和巡逻机器人：表现先遣建设阶段的自动化。

## 体验目标

第一版不做任务系统。玩家只需要能在基地附近走动，靠近建筑时看到标签，感受“SpaceX 到达火星之后，基地刚刚开始运转”的场景。
