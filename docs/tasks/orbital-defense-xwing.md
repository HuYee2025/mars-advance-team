# ARES X-Wing 外环防御任务

更新时间：2026-07-12

## 目标

在不把主游戏扩成独立太空战争游戏的前提下，加入一次完整的基地轨道防御体验：玩家在基地登上 `ARES X-Wing`，飞到火星外环，搜索并击碎随机 `2–7` 颗正在撞向基地的危险陨石；完成后可继续自由飞行，按 `Q` 自动返航。

## 当前已实现闭环

1. 战机默认锁定并隐藏。玩家首次穿过星际之门、完成虫洞坠落回到火星后解锁；红、蓝、黄三架战机随后并排停在基地南侧空旷起降区，靠近任意一架后按 `E` 登机。
2. 地面停放时 S 翼合拢、尾焰关闭并保持离地。按 `E` 后点亮尾焰并沿地表滑跑，随后同时沿火星大圆弧增加角度和高度，形成连续弧线起飞；约 `7.2s` 后抵达离地 `500` 的环形轨道。全过程只用同一第三人称追尾镜头，不做远景插播、翻转、掉头或相机状态切换；机腹始终朝向火星、机背始终朝向太空。任务开始后其余两架停放模型暂时隐藏，返航后恢复。
3. 默认第三人称采用远距轨道观景镜头且不显示准星；按 `C` 切到座舱前缘第一人称后才显示小型准星，再按一次恢复第三人称。
4. 每局随机生成 `2–7` 颗危险陨石，数量显示为 `THREAT 剩余/总数`。它们在起飞期间隐藏，进入战斗模式后也不直接出现在准星中心。HUD 右侧雷达显示相对方位；危险陨石以红色本体和上方细“结构完整度 / STRUCTURAL INTEGRITY”条识别，不显示环形外框。每颗目标结构完整度为 `8`。
5. 起飞后默认沿离地 `500` 的环形轨道向前运行：鼠标转向，`A/D` 左右飞行并带轻微侧身，`W/S` 垂直上/下飞行；`Shift / Ctrl` 为 `76 / 16 u/s` 临时调速，松开回到 `34 u/s`。机身与追尾相机使用同一个屏幕稳定坐标系，战机始终水平显示；上下和侧向飞行只改变环境相对掠过方向和战机位置，不让火星球面法线或纵向输入造成翻滚。松开移动键后再以限速方式平滑回归默认轨道。
6. 常规光枪每次命中削减 1 点结构完整度；按 `I` 锁定后，高能炮只对仍在屏幕内的目标有效，发射红色高能光束并一炮摧毁，使用后充能 `10s`。每次击毁奖励 `+10` 金币、`+50` 积分；全部清除后进入轨道巡航状态，不自动返航。
7. `Q` 在战斗或自由巡航任意阶段从当前位置连续自动导航到火星地面停机位；全程沿用当前追尾镜头和飞行姿态，不启动机翼收拢、镜头重置或独立返航表演。未完成任务时为主动中止，导航结束恢复地表控制。
8. 战斗开始后所有危险陨石共享全局 `115s` 倒计时；没有飞行/武器能量。时间归零立即失败；全部清除后停表并进入无限自由巡航。
9. 战机与火星、火星地表任意实体、任意普通陨石或危险目标陨石接触都立即爆炸，本轮失败；爆炸使用橙红 additive 闪光、发光碎片和短时点光源，不得用会遮住背景的透明深度球。爆炸约 `0.9s` 后直接回到基地重生点，不执行返航动画。
10. 普通和危险陨石复用确定性火星岩石生成器：大块低面多边形组成随机石质体，带轻微连续起伏和岩层明暗，不做尖刺、花瓣、圆形坑贴片或规则宝石轮廓。普通陨石按体积拥有 `1–6` 点结构完整度，全部可被光枪击碎；每击碎一颗奖励 `+50` 积分。普通陨石不减少 `THREAT` 数量，每次重新出动时恢复。
11. 第三人称追尾距离收至 `23`，让战机更清晰；普通灰色陨石数量从 `58` 提升至 `174`，并扩大轨道分布与大小随机差距。太阳盘面、光芒、光晕与近景纹理同步放大 `1.48` 倍。

## 技术落点

- 独立模块：`src/orbital-defense.ts`
- 主循环与交互接入：`src/main.ts`
- 专用 HUD：`index.html`、`src/style.css`
- 中英文界面：`src/i18n/catalog.ts`

轨道模式不复制第二个火星场景，而是复用现有 `scene`、真实火星球体和光照。战机、外环陨石、威胁目标、激光束与爆炸由独立 root 管理；任务结束后整体隐藏，避免污染地表循环。

## 当前边界

- 第一版每局只有随机 `2–7` 颗撞击威胁，不做敌机、舰队、武器升级或多星球地图；清除后只保留无敌人的无限自由飞行。
- 战机为项目内程序化原创低多边形模型，只借鉴 X 翼展开轮廓，不使用参考帖的模型或贴图。
- 最新模型按经典 X 翼结构重做：长尖六边形机鼻、深色座舱与框架、座舱后方宇航技工机器人、四片分离 S 翼、四台圆形引擎、四门翼尖激光炮和灰白旧化分区。三架地面实例分别用红、蓝、黄翼面涂装；飞行镜头、状态机和武器逻辑共用。
- `sFoilPivots` 同时驱动四片机翼和四组尾焰安装角：停放/滑跑为 `0` 合拢，爬升后段平滑过渡到 `1` 展开；停放模型的 `engineGlow` 必须保持隐藏。
- 任务定位为基地安全支线，可重复游玩；暂不阻断现有三章主线。
- 多人模式暂不同步战机和轨道战斗状态；轨道任务仍是各玩家本地独立进度。

## 第三人称镜头规则

- 第三人称不显示准星，镜头从战机切线后上方采用正常持续追尾；滑跑、爬升、战斗和返航共用完全相同的构图，避免为展示火星而突然抬高俯角、造成战机贴到画面上沿或发生跳切。
- 光枪仍沿战机机头方向发射，保留小范围瞄准辅助；第三人称镜头不再被当作精确瞄准视角。
- 战机必须完整显示，不允许镜头切进机身或只看到发动机尾焰。

## 第一人称视角规则

- `C` 只在轨道防御模式内切换视角，切换不改变战机姿态、飞行方向、准星射线或命中判定。
- 第一人称相机位于座舱前缘，沿战机 `local -Z` 精确朝前，使用更宽 FOV 表现速度感；HUD 的 `VIEW` 显示 `1P / 3P`。
- 第一人称才显示准星。命中判断先取得准星射线上的实际目标距离，左右机翼炮口向同一个落点汇聚，确保准星和弹着点一致。
- 每次登机、返航、失败恢复或重开任务都默认恢复第三人称，避免把轨道视角状态带回地表。
- 第三人称外部飞行不显示准星或任何陨石血条。第一人称仅对锁定或准星命中的危险陨石显示无文案的短细血条；普通陨石永不显示血条。
- `launch → combat` 的相机距离和高度用约 `1.6s` 连续插值，飞行速度和战机姿态沿用起飞末帧；基地场景不在切换点突然隐藏，避免画面跳切。

## 浏览器验收

- 红、蓝、黄三架战机均可靠近并按 `E` 登机，选择后轨道战机保留对应涂装。
- 地面状态机翼合拢、无尾焰且不陷地；登机后能看到点火、向前滑跑、爬升和展开机翼。
- 外环 HUD、火星、陨石带和战机追尾视角正常。
- 战斗开始时目标不在准星中心；右侧雷达红点能随战机转向改变方位，并可据此找到目标。
- 起飞中段战机始终留在画面内，镜头不会朝后、切到空场或在进入战斗时重置；战斗画面能看到完整火星。星点始终位于远景天幕，战机摄影机不会穿过点云。
- 按 `C` 可在 `1P / 3P` 之间往返切换，准星始终保持在正前方。
- `THREAT` 初始值在 `2/2–7/7` 之间；屏幕内危险陨石上方显示 `STRUCTURAL INTEGRITY` 细条且没有环形外框。
- `Space / J` 常规光枪命中后血条逐格下降；瞄准时按 `E` 锁定，按 `K` 出现红色高能光束并直接摧毁，随后状态提示约 `10s` 充能。
- 全部清除后仍可继续飞行，按 `Q` 后才自动返航。
- 返航后普通基地 HUD、建筑和互动恢复，提示可再次执行。
- `pnpm build` 与 `pnpm i18n:audit:strict` 通过。
