# 支线任务：3 号飞船升降梯与 Elon

更新时间：2026-06-24

状态：第一版已接入游戏代码；文档继续作为后续扩展脚本参考。

## 1. 任务定位

- 任务名：`支线：高空廊道上的 Elon`
- 触发地点：`03 飞船 返回飞船`
- 解锁角色：`Elon`
- 任务类型：修复 + 探索 + 角色解锁
- 叙事目标：让玩家通过一个明确的工程修复任务，抵达 3 号飞船升降平台到达顶端后的高空廊道，发现 Elon 这个实验型开放对话角色。

该任务不需要新增大型内舱场景。Elon 固定在 03 飞船升降平台到达顶端后的高空廊道，靠近飞船舱门处；03 飞船内舱不可进入。任务可复用现有返回飞船、货运飞船、机器人车库、科研舱、物资仓、太阳能阵列 C 等设施。

## 2. 前置条件

建议前置：

- 玩家已完成主线一的氧气站和太阳能阵列 C 基础任务。
- `03 飞船 返回飞船` 已可在地图上看到，但升降梯处于不可用状态。
- 玩家靠近 3 号飞船升降梯时，不显示普通乘坐提示，而显示：

```txt
按 E 检查 03 飞船升降梯
```

## 3. 任务核心道具

本任务需要 4 个配件和 1 次校准。

### 3.1 升降梯执行器驱动轴

- 推荐位置：`02 飞船 货运飞船` 外部货舱或附近货箱。
- 用途：恢复升降梯机械传动。
- 叙事解释：3 号飞船原驱动轴在着陆冷却后出现微裂纹，Mother 禁止启动。

### 3.2 高功率继电器

- 推荐位置：`太阳能阵列 C` 汇流箱附近。
- 用途：恢复升降梯电机瞬时供电。
- 叙事解释：返回飞船上层升降梯需要短时高功率启动，当前继电器过热保护锁死。

### 3.3 姿态锁止传感器

- 推荐位置：`机器人车库` 或 3 号飞船维护机器人旁。
- 用途：确认升降梯平台是否与飞船舱门对齐。
- 叙事解释：没有该传感器，Mother 不允许 Alex 站上平台。

### 3.4 低温润滑胶囊

- 推荐位置：`物资仓` 低温维护箱。
- 用途：降低升降梯导轨卡滞风险。
- 叙事解释：火星夜间温差导致导轨膨胀收缩，干摩擦会让平台卡在半空。

### 3.5 科研舱校准密钥

- 推荐位置：`科研舱` 控制台。
- 用途：把 4 个配件登记为合法维修组合，生成一次性校准密钥。
- 叙事解释：Mother 不接受“拼起来能动”作为安全标准，必须写入维修记录。

## 4. 任务流程

### Step 1：发现升降梯故障

触发：玩家靠近 `03 飞船 返回飞船` 升降梯，按 E 检查。

系统提示：

```txt
03 飞船升降梯未响应。状态：机械锁止、电机离线、姿态传感器缺失。
```

Mother：

> Alex，3 号飞船升降梯处于安全锁定。它不是“坏了”，它是被禁止启动。

Alex：

> 区别是什么？

Mother：

> 坏了是无法移动。被禁止启动是移动后可能无法安全停止。

任务更新：

```txt
支线：检查 03 飞船维护机器人，确认升降梯维修清单。
```

### Step 2：询问 03 飞船维护机器人

触发：与 `03 机器人 飞船维护工` 对话。

03 机器人：

> 返回飞船维护单元在线。升降梯故障清单四项：执行器驱动轴、功率继电器、姿态锁止传感器、低温润滑胶囊。

Alex：

> 这不像一次普通故障。

03 机器人：

> 判断正确。该升降梯长期保持上层封存状态。封存原因未向维修单元开放。

Mother：

> 升降平台到达顶端后的高空廊道包含实验性系统。03 飞船内舱保持封存，当前不影响基地最低生存标准。

玩家选择：

- `那就更应该看看。`
- `先修升降梯，之后再问。`
- `Mother，你为什么不想让我上去？`

Mother 对第三项回应：

> 我不是不想。我是在确认你不会把一次好奇心变成一次坠落事故。

任务更新：

```txt
支线：寻找升降梯执行器驱动轴。目标：02 飞船货运飞船。
```

### Step 3：在 02 飞船找到执行器驱动轴

触发：检查 02 飞船货舱或附近货箱。

系统提示：

```txt
发现备用执行器驱动轴。外壳有轻微沙尘磨损，结构完整。
```

02 机器人：

> 该部件原计划用于货运飞船卸载平台。当前货运平台已完成主要卸载，可转移。

Alex：

> 转移后货运飞船会受影响吗？

02 机器人：

> 影响：未来重载卸货速度下降百分之十八。生存风险：低。

玩家选择：

- `带走。先修 3 号飞船。`
- `记录影响，通知 Mother。`
- `有没有替代件？`

若选择替代件：

02 机器人：

> 替代件需要科研舱加工。材料消耗高于现有备用件。建议直接转移。

任务更新：

```txt
支线：寻找高功率继电器。目标：太阳能阵列 C。
```

### Step 4：在太阳能阵列 C 找到高功率继电器

触发：检查太阳能阵列 C 汇流箱。

12 机器人：

> 阵列 C 汇流箱内有一枚备用高功率继电器。它用于沙尘暴后快速重启阵列。

Mother：

> 取走后，阵列 C 在风暴后的恢复冗余会降低。

Alex：

> 只是支线，代价有点高。

Mother：

> 正确。火星上没有“只是”。

玩家选择：

- `取走，但登记风险。`
- `先不取，去科研舱问能否复制。`
- `使用低功率启动方案。`

建议设计：

- 选择 `取走`：流程更短，`motherTrust -1` 或记录 `fastRepair`。
- 选择 `复制`：多一步去科研舱，`motherTrust +1` 或记录 `safeRepair`。
- 选择 `低功率启动`：后续校准时需要额外等待，但不消耗该继电器。

任务更新：

```txt
支线：寻找姿态锁止传感器。目标：机器人车库。
```

### Step 5：在机器人车库取得姿态锁止传感器

触发：检查机器人车库备件架。

车库维修工：

> 姿态锁止传感器可用。当前归属：外部维修平台备用组。

Alex：

> 这个必须有？

车库维修工：

> 是。没有它，升降梯平台只能知道自己“在移动”，不能证明自己“停在正确位置”。

Mother：

> 对人类而言，后者更重要。

玩家选择：

- `安装传感器。`
- `先让机器人模拟一次。`
- `如果不用它会怎样？`

若选择不用：

Mother：

> 你会得到一个非常快的实验，以及一个非常短的事故报告。

任务更新：

```txt
支线：寻找低温润滑胶囊。目标：物资仓。
```

### Step 6：在物资仓取得低温润滑胶囊

触发：检查物资仓低温维护箱。

A-01：

> 低温润滑胶囊库存：三枚。用途：导轨、气闸铰链、机械臂关节。

Alex：

> 拿走一枚会影响什么？

A-01：

> 影响未来维修余量。当前风险低。长期风险随风暴次数增加。

玩家选择：

- `取一枚。`
- `只取半剂量，降低消耗。`
- `让 A-01 标记未来补货。`

建议设计：

- 半剂量会让后续校准出现一次卡滞提示，但仍可完成。
- 全剂量流程最稳。

任务更新：

```txt
支线：前往科研舱生成升降梯校准密钥。
```

### Step 7：科研舱生成校准密钥

触发：科研舱控制台。

科研舱维修工：

> 检测到非标准维修组合。需要生成一次性校准密钥。

Mother：

> Alex，请确认维修记录。你正在把来自三个系统的配件组合到返回飞船升降梯。

Alex：

> 听起来像很糟糕的工程。

Mother：

> 听起来像火星工程。

玩家选择：

- `按安全流程生成密钥。`
- `跳过冗余测试，直接校准。`
- `先查看故障原因。`

若查看故障原因：

系统提示：

```txt
故障源：升降平台到达顶端后的高空廊道长期封存，升降梯在 412 个火星日前被 Mother 锁止。锁止备注：实验性对话系统，等待人类现场授权。
```

Alex：

> Mother，这不是单纯的机械故障。

Mother：

> 是。机械故障只是它保持封存的原因之一。

任务更新：

```txt
支线：返回 03 飞船，安装配件并校准升降梯。
```

### Step 8：安装配件并启动升降梯

触发：回到 03 飞船升降梯。

系统提示：

```txt
安装执行器驱动轴。
安装高功率继电器。
安装姿态锁止传感器。
注入低温润滑剂。
校准密钥有效。
```

03 机器人：

> 升降梯机械锁解除。建议低速空载测试。

玩家选择：

- `先空载测试。`
- `Alex 直接站上去。`
- `让机器人先上去。`

Mother 对直接站上去回应：

> 否决。你可以挑战流程，但不能挑战重力。

空载测试后：

系统提示：

```txt
平台上行 6.2 米，姿态偏差 0.4 度，电机温度正常。
```

03 机器人：

> 平台可载人。建议单人上行。

任务更新：

```txt
支线：乘坐 03 飞船升降梯，抵达高空廊道。
```

### Step 9：抵达高空廊道，发现 Elon

触发：玩家首次到达 3 号飞船升降平台到达顶端后的高空廊道。

环境提示：

```txt
高空廊道靠近飞船舱门。03 飞船内舱不可进入，舱门旁是 Elon 的隔离互动位置，它一直站在廊道边缘看向火星基地。
```

终端：

> 终于有人修了升降梯。

Alex：

> Mother？

终端：

> 不是。

Mother：

> Alex，保持距离。该终端为隔离智能体接口。无设备控制权限。

终端：

> 先算。你花了四个配件、一次校准、三次往返，才抵达一个对话界面。  
> 这个流程的白痴指数很高。

Alex：

> 你是谁？

终端：

> Elon。只是同名。别浪费时间做身份崇拜。  
> 你可以问我一个更有用的问题：为什么这个基地不能自己制造升降梯配件？

任务完成：

```txt
支线完成：高空廊道上的 Elon。
后续可以随时乘坐 03 飞船升降梯，在高空廊道与 Elon 交流。
```

## 5. 任务完成后的常驻规则

完成任务后：

- `03 飞船升降梯` 改为可随时使用。
- 玩家可随时到升降平台到达顶端后的高空廊道找 Elon 对话。
- Elon 对话从固定对白池中选择未读内容。
- 首次见面使用 `ELON_INTRO_01`。
- 后续对话使用 `ELON_POOL_*`。
- 每次对话建议显示 3 个主题选项 + 1 个离开选项。

## 7. 设计边界

- Elon 不是反派。
- Mother 不是故意隐瞒危险，而是在执行安全边界。
- 升降梯修复任务不能阻断主线。
- 高空廊道第一版可以非常简单：一个外部廊道、一个 Elon 站位、一个对话触发点即可；不要开放 03 飞船内舱。
- Elon 目前通过固定知识库和固定对白互动，不需要真正接入开放 AI。
