# 远古巨树拱门选址草案

更新时间：2026-06-27

## 参考图结论

参考图的核心尺度关系不是普通建筑，而是“玩家在底部时，洞口顶部接近或超出视野上缘”的超级遗迹。

按当前游戏尺度估算：

- Alex 模型高度约 `2.3` 游戏单位。
- 03 飞船总高约 `45` 游戏单位。
- 通信塔约 `15` 游戏单位。
- 黑色方碑约 `5.9` 游戏单位。

推荐远古巨树拱门第一版尺寸：

- 总高：`118` 游戏单位，约 `51` 个 Alex，约 `2.6` 艘飞船高度。
- 外宽：`92` 游戏单位。
- 主洞开口高：`78` 游戏单位。
- 主洞开口宽：`42` 游戏单位。
- 根部占地半径：`58` 游戏单位。
- 材质方向：深灰石化木 / 玄武岩，纵向纤维纹理，局部破洞和不规则凹坑。

这个高度已经接近当前小星球半径 `PLANET_RADIUS = 132`，因此不建议第一版超过 `130`。超过后会像“插在小球上的塔”，而不是远古遗迹。

草图和基座范围：

- `docs/site-planning/ancient-tree-arch-sketch-and-footprint.svg`
- 草图包含正视、侧视、顶视和地图经纬度覆盖范围。
- 当前建议基座预留半径：`58` 游戏单位。
- 当前建议碰撞半径：`58-64` 游戏单位。

## 推荐位置

### 新推荐位置：用户标注点

- 半球：背面 / 黑暗半球，`normal.y < 0`
- 经纬度：约 `lon 0° / lat -50°`
- 投影坐标：约 `X 0 / Z 111`
- 对应法线：从经纬度 `lon 0° / lat -50°` 反算，约等于 `(0, -0.766, 0.643)`
- 建议 yaw：`0.0` 到 `0.35`，让拱门洞口朝向北侧/东北侧，方便玩家接近时看到完整开口。

距离估算：

- 距离居住舱约 `271` 游戏单位。
- 距离玩家出生点约 `378` 游戏单位。
- 距离福福坠毁飞船约 `209` 游戏单位。
- 距离旧候选点约 `96` 游戏单位。

选择理由：

- 位于背面黑暗区域，符合“星球背面远古巨物”的叙事。
- 比旧候选点更靠近黑暗半球中央，视觉上更像“孤立的远古遗迹”，而不是靠近地图边缘的普通远点。
- 离主基地更远，不和基地、太阳能阵列、三艘飞船争夺首屏视觉。
- 和福福坠毁飞船保持约 `209` 游戏单位距离，仍能区分为另一条神秘线索。
- 适合作为后续“远征/雷达发现/黑暗半球探索”的目标。

### 旧候选位置

旧候选位置：

- 半球：背面 / 黑暗半球，`normal.y < 0`
- 投影坐标：`X 210 / Z 260`
- 经纬度：约 `lon 38.9° / lat -21.6°`
- 距离居住舱约 `216` 游戏单位。
- 距离玩家出生点约 `346` 游戏单位。
- 距离福福坠毁飞船约 `267` 游戏单位。

旧候选点仍可保留为备用点，但当前优先级低于用户标注的新位置。

## 实现状态

2026-06-27 已按用户确认位置落入游戏场景：

- 源码位置：`src/world.ts`
- 实际位置：`lon 0° / lat -50°`，背面黑暗半球，投影约 `X 0 / Z 111`
- 模型方式：一体化挤出体，底部自然扩张、向上收缩，视觉上是“石化远古巨树树桩”，不再拆成独立基座和主体。
- 尺寸：总高 `118`、外宽 `92`、根部半宽/占地半径 `58`、洞口约 `42 x 78`、深度 `46`。
- 通路：门洞中间铺设一条 `24 x 116` 的深色道路，贴近地表，供人和车直接穿过。
- 碰撞：左右实体和根部使用多组球面 normal 碰撞体；实体部分无法穿过，中间门洞不放碰撞体，保持可通行。
- 系统适配：`CircleCollider` 支持 `normal`，玩家、足球、相机遮挡和车辆避障都可识别背面经纬度碰撞体；金币与基地平面机器人逻辑会跳过背面专用碰撞体，避免误判。
