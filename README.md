# 火星殖民 3D Demo

一个浏览器里的火星先遣基地视觉 Demo。第一版目标是做出可逛、可观察、氛围明确的小型火星殖民场景：登陆飞船、居住舱、温室、ISRU 工厂、太阳能阵列、机器人巡逻和沙尘环境。

## 当前版本

- 技术栈：Vite + TypeScript + Three.js。
- 资源方式：第一版不使用外部模型，全部用程序化几何生成。
- 体验形式：浏览器 3D 纯视觉探索 Demo。
- 操作：桌面端支持键盘移动和鼠标视角；移动端显示虚拟摇杆。
